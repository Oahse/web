## Screenshot Guidelines

### Dimensions
- Desktop screenshots: 1920x1080 or 1440x900
- Mobile screenshots: 375x812 (iPhone X) or 360x640 (Android)
- Tablet screenshots: 768x1024 (iPad)

### Format
- Use PNG format for static screenshots
- Use GIF format for animated workflows (max 5MB)
- Optimize images for web (use tools like TinyPNG)

### Content
- Use realistic sample data
- Blur or remove any sensitive information
- Show the application in a clean, professional state
- Include diverse product categories and user data

## Creating Screenshots

### Tools
- **macOS**: Cmd + Shift + 4 (area selection)
- **Windows**: Windows + Shift + S
- **Browser DevTools**: F12 → Device Toolbar for responsive views
- **GIF Recording**: LICEcap, Kap, or ScreenToGif

### Workflow GIFs to Create
1. **Customer Journey**: Browse → Add to Cart → Checkout → Order Confirmation
2. **Admin Product Creation**: Navigate to Products → Create New → Add Details → Save
3. **Order Management**: View Orders → Update Status → Send Notification
